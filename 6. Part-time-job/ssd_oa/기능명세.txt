2024-11-27 스스디

*기능명세
1. 사방넷 원본 엑셀 파일 다운로드
2. 해당 엑셀 파일 내 착불비 자동 입력
3. 상품코드를 vlookup으로 상품명 매칭 확인 (필수X)
4. 최종 데이터는 스스디/쇼블리스로 나누어 각 파일로 저장

5. 송장 출력 기능 (프린터 연동 A4 사이즈)
    - 바코드 X
    - 고객명, [상품번호] 상품명, 상품명2, 배달 주소
6. 배달지 주소에 따른 네이버 지도 API 지점 표기 기능


*흐름도
    - 착불비 입력 기능
        1. 엑셀 파일 다운 (수기)
        2. 엑셀 파일 읽기
            1) 원본 파일을 라인별로 읽음
            2) 상품 기준으로 착불비 입력된 기준 파일
            3) 우편번호 기준으로 수도권/지방 등 입력된 파일
            2.3 파일과 비교해서 착불값 입력
        3. 원본 엑셀 파일 내 착불비 Column 추가 후 계산값 입력
        4. 파일은 스스디/쇼블리스 나누어 각각 저장됨

    - 송장 출력 기능
        1. 엑셀 파일 읽기
            1-1. 전체 엑셀 파일 읽은 후 체크박스로 선택하기
            1-2. 출력이 필요한 데이터만 추린 후, 엑셀 파일로 읽어서 처리
        2. 출력하기
            2-1. 체크박스 선택한 데이터를 읽어서 프린터 양식 만들기
            2-2. 전체 읽어서 프린터 양식 만들기
    
    - 우편번호를 기준으로 네이버 API 지점 표기 기능
        1. 엑셀 파일 읽기
        2. 우편번호를 기준으로 네이버 API 지도에 점으로 표시하여 png 파일로 추출
    (2024.12.12 확인) 웹 URL을 등록하여 restAPI를 호출하는 구조로 현재 구조에서는 사용 불가함
